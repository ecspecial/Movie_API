Movie_API Documentation page

This API is inentended to listen for requests and do the following:

Business logic Endpoint URL Method type Query parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET - - JSON objects containing all movies
Return movie information by title /movies/:title GET :title - JSON object with data on single movie
Return information about movie genre /movies/genre/:genreName GET :genreName - JSON object cintaining data about genre
Return data about a director by name /movies/directors/:directorName GET :directorName - JSON object with data about director
Allow new users to register /users POST - JSON object holding user data, format: { "name" : "Summer", "favorite movie" : [] } JSON object holding added user data , format: { "name" : "Summer", "favorite movie" : [] } with ID
Allow users to update their user info (username) /users/:id PUT :id - JSON object holding updated user data
Allow users to add a movie to favorites POST /users/:id/:movieTitle :movieTitle - Message and status indicating whether movie was added
Allow users to remove a movie from favorites DELETE /users/:id/:movieTitle :movieTitle - Message and status indicating whether movie was deleted
Allow existing users to deregister DELETE /users/:id :id - Message and status indicating whether user was deleted
